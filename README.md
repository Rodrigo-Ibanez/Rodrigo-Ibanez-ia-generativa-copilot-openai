# Explorando os Recursos de IA Generativa com Copilot e OpenAI

Este repositório foi criado para explorar os recursos de **IA Generativa** utilizando o **Copilot** e a **API OpenAI**. O foco do projeto é realizar o reconhecimento de texto em imagens utilizando técnicas de OCR (Reconhecimento Óptico de Caracteres).

## Passo a Passo

### 1. Criando as Pastas
Foram criadas duas pastas principais no repositório:
- **inputs**: Contém as imagens utilizadas para o reconhecimento de texto.
- **output**: Armazena os resultados do reconhecimento de texto nas imagens.

### 2. Processamento das Imagens
Para realizar o reconhecimento de texto nas imagens, utilizamos o recurso **OCR**. Cada imagem da pasta `inputs` foi processada, e o texto extraído foi salvo na pasta `output` como arquivos de texto.

### 3. Ferramentas Utilizadas
- **Copilot**: Auxiliou no desenvolvimento de partes do código.
- **OpenAI**: API utilizada para entender como a IA pode gerar ou processar texto.


### 4. Insights Adquiridos
Ao utilizar a IA generativa e ferramentas de OCR, foi possível perceber que a combinação de diferentes tecnologias pode acelerar significativamente o processo de extração e interpretação de dados a partir de imagens.

### 5. Possibilidades Futuras
Com as ferramentas de IA, várias melhorias podem ser feitas, como:
- Reconhecimento de texto em diferentes idiomas.
- Identificação de informações específicas dentro de imagens (como números, nomes, ou locais).
- Integração com outras tecnologias para análise de sentimentos ou classificações automáticas.
